# Airbnb

- **App Type:** Travel & accommodation marketplace (content browsing)
- **Tags:** travel, marketplace, photo-forward, minimal, warm-neutrals, booking, content-browsing
- **Platform:** Mobile-first (375px)
- **Interaction Density:** Medium

## Screens

| File | Pattern | Description |
|------|---------|-------------|
| `single-screen.html` | Single Screen | Explore homepage with pill search bar, horizontal category strip, listing cards with gradient photo placeholders, bottom tab navigation |
| `multi-state-screen.html` | Multi-State | Listing detail page with 6 states: default, dates selected, guest selection, reserve confirmation, share sheet, photo gallery |
| `celebration-screen.html` | Celebration | Booking confirmed with confetti burst, animated checkmark, reservation card, staggered stat reveals, confirmation banner |

## Key Patterns

- Pill-shaped search bar with segmented sections (location, dates, guests)
- Horizontal scrolling category strip with icon + label and active underline
- Listing cards with gradient photo placeholders, heart overlay, dot indicators, and price/rating metadata
- Bottom tab navigation with icon + label and active coral highlight
- Booking bar fixed to bottom with price and reserve CTA
- Guest counter sheet with +/- stepper controls
- Price breakdown with line items and total
- Share sheet with app icon grid
- Dark fullscreen photo gallery with navigation arrows
- Confetti particle burst with coral/gold/green palette
- SVG checkmark draw-on animation with bounce
- Staggered fade-up stat cards with animation delays
