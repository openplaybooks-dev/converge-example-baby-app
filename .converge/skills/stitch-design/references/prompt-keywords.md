# UI/UX Keywords Reference for Flutter

Progressive disclosure reference for Flutter UI terminology and adjective palettes.

## Flutter Widget Keywords

### Navigation
- AppBar, SliverAppBar, NavigationBar
- NavigationDrawer, NavigationRail
- TabBar, TabBarView, DefaultTabController
- BottomSheet, showModalBottomSheet
- BackButton, CloseButton, PopScope

### Content Containers
- Card, Card.outlined, Card.filled
- ListTile, ExpansionTile
- CustomScrollView, SliverList, SliverGrid
- PageView, CarouselView
- Dialog, AlertDialog, SimpleDialog

### Forms
- TextField, TextFormField
- DropdownButton, DropdownMenu
- Checkbox, Radio, Switch
- Slider, RangeSlider
- DatePicker, TimePicker
- SearchBar, SearchAnchor

### Calls to Action
- ElevatedButton, FilledButton, FilledButton.tonal
- OutlinedButton, TextButton
- FloatingActionButton, FloatingActionButton.extended
- IconButton, CloseButton, BackButton

### Feedback
- SnackBar, MaterialBanner
- CircularProgressIndicator, LinearProgressIndicator
- RefreshIndicator
- Shimmer (skeleton loading)
- Badge, Tooltip

### Layout
- Scaffold, SafeArea
- Column, Row, Wrap, Stack
- Expanded, Flexible, Spacer
- SizedBox, Padding, Center, Align
- ConstrainedBox, AspectRatio

## Adjective Palettes

### Minimal / Clean
- generous EdgeInsets, breathing room
- surfaceContainerLowest backgrounds
- subtle outlineVariant borders
- single-weight typography

### Professional / Corporate
- neutral ColorScheme seed
- consistent elevation hierarchy
- structured GridView layouts
- monospace labelSmall for data

### Playful / Fun
- vibrant primary seed color
- large radiusLg on cards
- Curves.elasticOut animations
- bold headlineLarge weights

### Dark Mode
- dark ColorScheme brightness
- surfaceContainer for elevation
- high-contrast onSurface text
- subtle outline borders

## Color Role Terminology (Material 3)

### Surfaces
- `surface` — default background
- `surfaceContainer` — slightly elevated
- `surfaceContainerHigh` — more elevated
- `surfaceContainerHighest` — most elevated
- `inverseSurface` — inverted for contrast

### Content
- `onSurface` — primary text
- `onSurfaceVariant` — secondary text
- `outline` — borders, dividers
- `outlineVariant` — subtle borders

### Accent
- `primary` — brand, CTAs, active states
- `secondary` — supporting accent
- `tertiary` — complementary accent
- `error` — destructive, validation errors
